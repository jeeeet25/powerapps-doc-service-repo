# Attach your Power Automate Flow export here

## What to place in this folder
- Exported **Package (.zip)** from Power Automate:
  - `YourFlowName.package.zip`

## Export steps (Power Automate)
1. Open the flow
2. Export → Package (.zip)
3. Download
4. Copy into this folder

> If you have child flows, create a subfolder `child_flows/` and store each package zip there.
